## Photo requirements

| Property | Recommendation |
|---|---|
| Format | JPEG |
| Aspect ratio | 4 : 5 (portrait) |
| Resolution | 800 × 1000 px |
| File size | under 300 KB |
| Color space | sRGB |
| Subject framing | head and shoulders, face centered |

## Notes

- If a file is missing the page shows a striped placeholder — no error.
- `object-fit: cover` crops to fill the 4:5 frame, so slightly off-ratio photos
  are fine. Avoid landscape shots — they will be heavily cropped.
- Quickest workflow: crop to 4:5 in Photos or Lightroom, export JPEG at
  medium-high quality (~85%), rename to the filename above.
